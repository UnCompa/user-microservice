# Microservicio de Usuario

## Description

Creación de un microservicio de usuario

## Project setup

Usando NPM

```bash
npm install
```

Usando PNPM

```bash
pnpm install
```

## Copilar y ejecutar el proyecto

Con NPM

```bash
# Desarrollo
$ npm run start

# Modo watch
$ npm run start:dev

# Construir aplicación
$ npm run build

# Produccion
$ npm run start:prod
```

Con PNPM

```bash
# Desarrollo
$ pnpm run start

# Modo watch
$ pnpm run start:dev

# Construir aplicación
$ pnpm run build

# Produccion
$ pnpm run start:prod
```

## Levantar tests

Con NPM

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

Con PNPM

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```

## Arquitectura

La arquitectura usada esta basada en la hexagonal, adaptada para la creacion de microservicios
